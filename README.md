# Intelligent Message Search System for VK WorkSpace

## Обзор проекта
Высокопроизводительный поисковый движок на базе архитектуры RAG (Retrieval-Augmented Generation), оптимизированный для работы в изолированном контуре корпоративной сети. Система предназначена для обеспечения релевантного поиска по фрагментированным данным мессенджера.

## Эффективность системы
В ходе разработки было достигнуто значительное улучшение метрик относительно базового решения.

*   **Базовый балл (Baseline)**: 0.4600
*   **Итоговый балл (Final Score)**: 0.5707
*   **Recall@50**: 0.5881
*   **nDCG@50**: 0.5013

## Технологический стек
*   **Векторное хранилище**: Qdrant
*   **Модели**: Qwen3-Embedding (Dense), BM25 (Sparse), Llama-Nemotron (Reranker)
*   **определение архитектуры**: Двухстадийный пайплайн с ансамблем из 4-х потоков поиска и алгоритмом диверсификации результатов NDCG Sharpener.
