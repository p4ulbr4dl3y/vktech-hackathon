# Введение: Интеллектуальный поиск VK WorkSpace

Система представляет собой высокопроизводительный поисковый движок на базе архитектуры **RAG (Retrieval-Augmented Generation)**, специально оптимизированный для работы с корпоративными данными мессенджера.

## Проблема
Пользователи корпоративных чатов часто ищут информацию в огромных массивах переписки. Обычный полнотекстовый поиск часто не справляется из-за:
1.  **Фрагментарности**: Ответ разбит на несколько коротких сообщений.
2.  **Контекста**: Важно не только слово, но и кто, когда и в каком чате его написал.
3.  **Синонимии**: Пользователь спрашивает "как настроить почту", а в чате написано "инструкция по конфигурации SMTP".

## Наше решение
Мы внедрили систему, которая понимает **семантику** вопроса. Наш движок не просто ищет совпадения слов, а анализирует смысл сообщения и контекст обсуждения.

### Ключевые достижения (Рекорд хакатона)
*   **Score**: 0.6014 (при бейзлайне 0.46)
*   **Recall@50**: 0.6225 (мы находим правильный ответ в 62% случаев среди первых 50 результатов).
*   **nDCG@50**: 0.5173 (правильные ответы находятся максимально близко к началу списка).
